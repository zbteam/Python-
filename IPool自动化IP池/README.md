# IPool
IPool  自动化IP池

每次运行会自动获取最新IP，什么时候使用什么时候打开

## IP代理池作用

1.渗透过程中IP被封，可换IP继续操作

2.可以刷票，批量注册，如ip验的站

3.可以验证有效代理访问xx网站等等
****
输入要验证的网址，相当于这些ip可以访问哪个网址

扫描完成后会显示可用IP列表，同时自动保存到success_ip.txt
